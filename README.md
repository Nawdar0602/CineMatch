# CineMatch

CineMatch is een sociaal film discovery platform dat de kracht van vriendschapsaanbevelingen combineert met een intuïtieve swipe-interface. Gebruikers ontdekken films via de reviews en ratings van vrienden, met een verslavend swipe-systeem dat het ontdekken leuk maakt.

## Kernfunctionaliteiten

1. **Swipe-Interface**
   - Rechts swipen: Like
   - Links swipen: Dislike
   - Omhoog swipen: Super Like
   - Tap: Filmdetails bekijken

2. **Matchsysteem**
   - Pop-up bij gedeelde likes met vriend(in)
   - Smaakmatch met Compatibility Score

3. **Live Zoeken & Filteren**
   - Zoek op titel, genre, regisseur of acteur
   - Diverse filteropties

4. **Watchlist Management**
   - Slim gesorteerd op prioriteit
   - Status-indicatoren
   - Streamingchecker

5. **Sociale Features**
   - Reviews met 5-sterren systeem
   - Mood-tags
   - Vriendennetwerk

6. **Inlogsysteem & Gebruikersbeheer**
   - Registratie & Inloggen
   - Profielbeheer
   - Back-end Validatie

## Technische Stack

### Front-end
- React + TypeScript
- Tailwind CSS (mobile-first)
- Headless UI voor modals & menu's
- Framer Motion voor animaties
- Zustand voor state management
- React Router voor pagina-navigatie

### Back-end
- PHPMyAdmin
- Node.js waar nodig
- TheMovieDB API

## Installatie

```bash
# Installeer dependencies
npm install

# Start de ontwikkelserver
npm start
```
